# Learn Python

Esse repositório contém recursos e materiais para aprender Python, uma linguagem de programação versátil e amplamente utilizada.
Os materiais foram retirados do DataCamp e Kaggle( No futuro pode ser adicionado mais materiais de outras plataformas). A ideia principal é todos os exemplos e exercicios das plataformas estarem aqui para facilitar o acesso e o estudo.

## Requisitos

- Python 3.x instalado no seu computador.
- Editor de texto ou IDE (como VSCode, PyCharm, Jupyter Notebook, etc.).


## Como usar este repositório
- Clone ou faça o download deste repositório para o seu computador.
- Navegue pelos diretórios para encontrar os materiais de estudo.
- Ative o ambiente virtual (se aplicável) para garantir que todas as dependências estejam instaladas.
